# **App Name**: ShikshaSphere

## Core Features:

- Curriculum Mapping: Map the STEM curriculum of Odisha state to interactive learning modules.
- Interactive Modules: Offer interactive learning modules with quizzes, simulations, and challenges.
- Gamified Progress Tracking: Track student progress through points, badges, and leaderboards to enhance engagement.
- AI-Powered Tutoring: Use an AI tool to offer personalized guidance based on individual learning styles and knowledge gaps.
- Odia Language Support: Provide all content and interactions in Odia, the local language, to improve accessibility.
- Cultural Integration: Integrate elements from Odisha's culture, history, and local innovations into the learning modules.
- Adaptive Difficulty: Dynamically adjust the difficulty level of questions based on student performance to provide optimal challenge.

## Style Guidelines:

- Primary color: Vibrant saffron (#FF9933), reminiscent of the Indian flag and associated with knowledge and sacrifice.
- Background color: Light beige (#F2E9E1), a desaturated tone of the primary, provides a warm, neutral background.
- Accent color: Muted teal (#339980), analogous to saffron but significantly different in brightness and saturation, adds a refreshing contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif font for a modern, friendly, and readable style. Note: currently only Google Fonts are supported.
- Use line-art icons depicting educational concepts, achievements, and cultural elements.
- A clean and spacious layout with a clear visual hierarchy to present content and avoid overwhelming students.
- Use subtle transitions and animations to provide feedback, reinforce achievements, and make the app engaging.
